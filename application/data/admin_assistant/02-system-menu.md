# System configuration map / 系统菜单导览

**更全的配置页说明**：`10-system-configuration-menus.md`。**侧栏其它模块一览**：`11-menu-modules-content-users.md`。**本站目录结构**：`23-directories-and-runtime.md`。

Below lists **common** MacCMS admin menu names. Your build may group them slightly differently; use the left sidebar and search.

## 系统 (System)

| Menu (常见名称) | Purpose |
|-----------------|---------|
| **网站参数 / 基本设置** | 站点名、域名、安装路径、ICP、统计代码、注册开关、上传限制、时区语言等。 |
| **站外入库设置** | 接收远程推送入库的接口配置（与 API 对接相关）。 |
| **API 设置** | 对外提供数据接口、密钥、分类映射等（供 APP/采集端使用）。 |
| **URL 设置** | 路由、伪静态规则、各模块 URL 模式（视频/文章/专题等）。 |
| **SEO 设置** | 首页与栏目 TDK、sitemap、推送相关。 |
| **附件设置** | 本地上传、远程附件、云存储（若已对接）。 |
| **会员设置** | 会员中心、积分、订单、试看等（若启用会员）。 |
| **邮件 / 短信** | 通知、验证码（若配置）。 |
| **采集参数配置** | 全局采集默认值：审核状态、随机点击、同步图片、同义词等（**先配好再大批量采集**）。 |
| **AI 搜索配置** | 前台 AI 搜索/对话、外部数据源、语义重排、速率限制等。 |
| **后台助手配置** | 浮动 **`?`**、知识库 `application/data/admin_assistant/*.md`、助手专用或复用 AI 搜索的 LLM 凭据。 |
| **URL 推送** | 向搜索引擎推送 URL（按版本功能为准）。 |
| **定时任务** | 说明或跳转至 `api.php` 定时访问说明（具体以界面为准）。 |
| **网站安全** | 防注入、违禁词、敏感过滤等。 |
| **开放分类 / 扩展字段** | 视频/文章扩展分类、扩展状态等（与前台筛选相关）。 |
| **播放器 / 下载器 / 服务器组** | 解析播放、多线路、下载方式（视频核心）。 |
| **管理员 / 角色 / 权限** | 后台账号与菜单权限。 |

## 视频 / 文章 / 专题 / 漫画 等

- **数据列表**：审核、推荐、锁定、批量删除/修改分类。  
- **添加**：手工单条发布。  
- **批量操作 / 重复数据**：维护质量与去重。  

## 采集

- **自定义资源 / 自定义规则**：绑定资源站 API 或规则，**分类映射**到本地类型。  
- **断点 / 当天 / 分类采集**：按节点执行同步。  
- **采集片库**（若存在）：片库型资源维护。  

## 模板

- 选择 **PC / WAP 模板目录**，在线编辑或 FTP 修改 `template/` 下文件。  
- 修改含 JS/CSS 的模板后，建议 **清理缓存** 并强刷浏览器。

## 数据库 / 备份

- **SQL 执行 / 替换**：高危操作，仅在有备份且理解语句时使用。  
- **Meilisearch / 搜索索引**（若启用）：全量或增量同步与后台入口以当前版本为准。

## 中文速查

- **改站点名、域名、统计代码** → **系统 → 网站参数**。  
- **改视频/文章列表网址形态** → **系统 → URL 设置** + Web 服务器 rewrite。  
- **采集入库默认是否审核、是否同步图** → **系统 → 采集参数配置**。  
- **前台 AI 搜索/对话** → **系统 → AI 搜索配置**。  
- **后台「?」助手** → **系统 → 后台助手配置**（可与 AI 搜索共用 Key）。  
- **新建文章/视频无分类** → 先到 **分类管理** 建好 **类型** 与 **分类**，再在内容里选择。

若用户找不到某项，请其说明 **左侧一级菜单名称** 或截图，不要猜测不存在的子菜单路径。
