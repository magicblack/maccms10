# Types, topics, manga / 分类、专题与漫画

## 1) 分类与类型（视频 / 文章 / 漫画等）

1. 进入 **视频 → 分类**、**文章 → 分类** 或统一的 **基础 → 分类**（以您后台为准）。  
2. **先建「类型」**（大模块），再在类型下 **新建子分类**。  
3. 每个分类可设置：**名称、排序、SEO 标题关键词描述、图标** 等（以表单为准）。  
4. **保存后**，再到添加内容页的 **下拉框** 中选择该分类。  
5. 若改了分类结构，前台 **缓存/静态页** 需更新。

错误排查：**添加内容时下拉无分类** → 未建分类或管理员无该分类权限（若权限细分到栏目）。

---

## 2) 专题（TOPIC）

1. **专题 → 添加专题**（或类似入口）。  
2. 填写 **专题名称、封面、简介**。  
3. **关联内容**：按界面引导 **勾选视频 ID / 文章 ID** 或按规则选取（以实现为准）。  
4. 保存后，前台通过 **模板中专题列表/专题详情标签** 展示；链接形态见 **URL 设置 → 专题**。

---

## 3) 漫画（若启用）

1. **漫画 → 分类** 与 **漫画 → 添加** 与视频/文章类似。  
2. **章节 / 阅读地址** 按表单说明填写（多源、多话数时常用特定分隔格式）。  
3. 若对接 **Meilisearch**，漫画可能参与 **独立索引**；改库后按后台说明 **同步索引**。  

---

## English summary

Create **parent type → child category** before adding content. **Topics** bundle VOD/ART entries. **Manga**: categories + chapters formatting per admin form hints.
