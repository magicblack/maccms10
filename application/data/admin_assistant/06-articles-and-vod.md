# How to create articles and videos / 如何发布文章与视频

## A. 新建文章（手工）

**前提**：已在 **文章 → 分类**（或 **基础 → 分类**）中创建 **文章类型** 与 **分类**。

### 步骤

1. 登录后台 → 左侧 **文章**（图标或文字可能为「文章管理」）。  
2. 打开 **添加文章**（或 **文章 → 添加**）。  
3. 必填项通常为：**标题**，**归属分类**，**正文或简介**（以表单为准）。  
4. **封面图**：上传或填写图片地址；注意附件/远程图设置是否允许外链。  
5. **TAG / 关键词**：可选；利于前台检索与聚合。  
6. **发布状态 / 审核**：若站点要求审核，先保存为「未审核」，由有权限用户在 **文章列表** 中批量或单条审核。  
7. 点击 **保存**。  
8. 前台打开 **文章分类页 / 详情页 URL**（取决于 **URL 设置** 与模板）核验展示。

### 文章采集入库

若文章来自采集：在 **采集** 中为文章节点配置好 **映射到本地文章分类**，执行采集后仍在 **文章 → 文章数据** 中检查、纠错、补图。

## B. 新建视频（手工）

**前提**：已配置 **视频类型/分类**，并已准备 **播放器**（**系统 → 播放器**）。

### 步骤

1. 后台 **视频 → 添加**（或 **视频 → 添加视频**）。  
2. 填写 **标题、分类、封面、简介** 等基础信息。  
3. **播放地址**：按播放器要求填写（多集常用分隔符如 `$$$` 等，以您后台输入说明为准）。  
4. **播放组 / 线路 / 解析**：若使用多组播放，在表单对应区域添加；与 **播放器代码** 一致。  
5. **演员、导演、地区、年代、扩展分类**：按运营需要填写，影响前台筛选。  
6. **状态**：上架/推荐/审核 等按权限操作。  
7. **保存** → 前台 **视频详情页** 验证播放与清晰度。  

### 视频批量维护

在 **视频 → 视频数据** 使用筛选、批量改分类、批量删除；大批量前务必备份或使用「仅一批测试」策略。

### 剧情 / 分集（若启用）

- 后台可能存在 **视频 → 有分集剧情** 等列表入口（`menu/vod_data_plot`）；用于维护 **连续剧分集文案或关联**。  
- **系统 → URL 地址配置** 中若有 **剧情** 路由，前台会有对应模板页；请以当前版本表单字段为准填写 **分集名称、内容与关联正片**。  
- 播放器、多集地址仍遵循 **播放器管理** 与 **`$$$` 分隔** 等约定（详见 **`13-player-server-downer.md`**）。

## C. URL 打不开或 404

1. **系统 → URL 设置**：确认模块路由与 PATHINFO / rewrite 一致。  
2. **Web 服务器**：Nginx / Apache **伪静态规则**需与官方示例一致（含 `try_files` / `RewriteRule`）。  
3. **分类是否隐藏、内容是否下架/未审核**。

## English summary

**Article**: Prepare **taxonomy** → **Article → Add** → fill title, category, body, thumb → save → verify front URL.  
**Video**: Prepare **taxonomy + player** → **Video → Add** → play URLs/lines → save → verify detail page playback.
